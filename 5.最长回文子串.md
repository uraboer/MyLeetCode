## 5.最长回文子串

### 题目

给定一个字符串 s ，找出 s 中的最长回文子串。可以假定 s 的最大长度为1000

> 示例1：
>
> Input : “babad”
>
> Output : “bab”



> 示例2：
>
> Input  : “cbbd”
>
> Output : “bb”

### 分析

回文的特点就是正着和反着是相同的，或者说是镜面的。所以用 sr 表示倒叙后的 s 字符串，如果是回文子串的一个充要条件就是它存在 sr 中且本身为回文



```python
def longestPalindrome(self,s):
    sr="".join(reversed(s))
    answerLen=1
    try:
        answer=s[0]
    except:
        return None
    i=0
    while i<len(s)-1:
        plus=2
        while sr.find(s[i:i+plus])!=-1 and plus<=len(s)-i:
            plus=plus+1
        if plus-1>answerLen:
            taskAnswer=s[i:i+plus-1]
            if taskAnswer==taskAnswer[::-1]:
                answer=taskAnswer
                answerLen=len(answer)
        i=i+1
    return answer
```
