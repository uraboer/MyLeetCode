## 4.两个排序数组的中位数

### 题目

给定两个大小为 m 和 n 的有序数组 nums1 和 nums2

找出这两个有序数组的中位数，要求算法的时间复杂度为 O(log(m+n))

> 实例1：
>
> nums1=[1,3]
>
> nums2=[2]
>
> 中位数是 2.0



> 示例2：
>
> nums1=[1,2]
>
> nums2=[3,4]
>
> 中位数是 (2+3)/2=2.5



```python
class Solution:
    def findMedianSortedArrays(self, nums1, nums2):
        """
        :type nums1: List[int]
        :type nums2: List[int]
        :rtype: float
        """
        res=nums1+nums2
        res.sort()
        n=len(res)//2
        return res[n] if len(res)%2==1 else (res[n]+res[n-1])/2
```
